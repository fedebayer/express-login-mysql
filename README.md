## Installation

```bash
git clone https://github.com/fedebayer/express-login-mysql
cd express-login-mysql
-- add enviroment variables in a new .env file --
npm i
npm run dev
```

### Environment Variables

* PORT=3000
* DB_HOST=yourhost
* DB_PORT=3306
* DB_USER=youruser
* DB_PASSWORD=yourpassword
* DB_DATABASE=yourdatabase
